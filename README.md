# Interior Design Web App

A responsive web application that showcases interior design layouts, room galleries, and dynamic content using Vanilla JavaScript, HTML, and CSS.

The project was developed to explore browser fundamentals including DOM manipulation, AJAX, cookies, local storage, dynamic content rendering, and JSON-driven UI generation.

---

## Features

- 🏠 Interactive home page
- 🖼️ Dynamic image gallery
- 📦 JSON-driven content rendering
- 🔄 AJAX data loading
- 🍪 Cookie management
- 💾 Local Storage support
- 🌐 Browser detection
- 📝 Dynamic form generation
- ⚡ DOM manipulation without frameworks

---

## Technologies Used

- HTML5
